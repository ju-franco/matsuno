# 🍱 Matsuno – Website Acadêmico  

[**Acesse O Site**](https://ju-franco.github.io/matsuno/)

Um website desenvolvido com fins acadêmicos para apresentar o restaurante japonês fictício **Matsuno**, destacando sua história, missão, visão, produtos e parceiros.  

Este projeto foi criado no **1º ano do curso técnico em Desenvolvimento de Sistemas**, como parte da disciplina **Programação Web (PWI)**.

---

## 📘 Sobre o Projeto

O objetivo do site é proporcionar uma experiência completa e visualmente atrativa ao usuário, simulando um restaurante japonês moderno.  
Nele, o visitante encontra:

- História, missão, visão e valores do restaurante  
- Galeria completa de produtos e combos  
- Menu dinâmico de pratos  
- Área de contato com informações e formulário  
- Seção especial exibindo os parceiros do projeto  
- Layout responsivo com efeitos visuais e navegação intuitiva  

O site foi desenvolvido inteiramente com HTML, CSS e JavaScript, utilizando estilizações personalizadas e plugins externos para animações e sliders.

---

## 🗂️ Estrutura do Site

O projeto contém as seguintes páginas:

- **index.html** – Página principal com banner, sobre o restaurante, produtos e parceiros  
- **contato.html** – Página dedicada com informações de contato e formulário  
- **style.css** – Estilos principais utilizados em todo o site  
- **css/responsive/** – Arquivos de responsividade  
- **footer/** – Estilos específicos do rodapé  
- **js/** – Scripts de animação, sliders, pré-carregamento e mapa  
- **img/** – Imagens de banners, pratos, logos, fundo, ícones e redes sociais  

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura das páginas  
- **CSS** – Estilização principal  
- **JavaScript** – Interações e animações  
- **Bootstrap** – Componentes e responsividade  
- **Google Fonts** – Tipografias personalizadas  
- **Font Awesome** – Ícones  

---

## 📄 Finalidade

Este projeto é **exclusivamente didático**, desenvolvido para prática de criação de websites usando HTML, CSS e JavaScript, aplicado ao tema fictício do restaurante Matsuno.
